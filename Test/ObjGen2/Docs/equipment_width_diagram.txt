EQUIPMENT WIDTH STRIP DIAGRAM
=============================

This diagram shows how a start bin relates to the entire strip of bins
when accounting for equipment width (15ft = 7.5 bins at 2ft per bin).

TOP VIEW - Equipment Cutting Strip:
===================================

Direction of Travel: →
Equipment Width: 15ft (7.5 bins)
Bin Size: 2ft x 2ft

    ┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐
    │  W  │  W  │  W  │  W  │  W  │  W  │  W  │  W  │  ← Equipment Width Strip
    │ -3  │ -2  │ -1  │  0  │ +1  │ +2  │ +3  │ +4  │  (Bin offsets from center)
    └─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘
         │     │     │     │     │     │     │
         │     │     │     │     │     │     └─ +4 bin (rightmost)
         │     │     │     │     │     └─ +3 bin
         │     │     │     │     └─ +2 bin
         │     │     │     └─ +1 bin
         │     │     └─ START BIN (center, w=0)
         │     └─ -1 bin
         └─ -2 bin (leftmost)

EQUIPMENT WIDTH CALCULATION:
============================

Equipment Width: 15ft = 4.572m
Bin Size: 2ft = 0.6096m
Equipment Width in Bins: 15ft ÷ 2ft = 7.5 bins
Rounded Up: 8 bins (EquipmentWidthBins = 8)

STRIP GENERATION ALGORITHM:
===========================

For each cutting direction (0° to 360° in 5° increments):

1. Calculate perpendicular direction:
   - PerpX = -sin(angle)
   - PerpY = cos(angle)

2. Generate strip bins:
   for (int w = -EquipmentWidthBins/2; w <= EquipmentWidthBins/2; w++)
   {
       StripBinX = CenterBinX + (w * PerpX)
       StripBinY = CenterBinY + (w * PerpY)
   }

3. Bin offsets from center:
   w = -4: Leftmost bin
   w = -3: 
   w = -2:
   w = -1:
   w =  0: START BIN (center)
   w = +1:
   w = +2:
   w = +3:
   w = +4: Rightmost bin

EXAMPLE: North-South Cutting (0°)
=================================

Direction: North (0°)
Perpendicular: East (90°)
PerpX = -sin(0°) = 0
PerpY = cos(0°) = 1

Strip extends EAST-WEST:
    W  W  W  W  S  W  W  W  W
   -4 -3 -2 -1  0 +1 +2 +3 +4
    │  │  │  │  │  │  │  │  │
    └──┴──┴──┴──┴──┴──┴──┴──┘
    ←── Equipment Width (15ft) ──→

==========================================

Direction: Northeast (45°)

PerpX = -sin(45°) = -0.707
PerpY = cos(45°) = 0.707

Strip extends NORTHWEST-SOUTHEAST:

VOLUME CALCULATION:
==================

Each bin in the strip contributes:
Volume = DepthM × 0.3716 m² (2ft × 2ft bin area)

Total Strip Volume = Σ(DepthM × 0.3716) for all bins in strip

KEY POINTS:
===========

1. START BIN is always the CENTER of the equipment width strip
2. Equipment processes ALL bins in the strip simultaneously
3. Strip orientation is PERPENDICULAR to cutting direction
4. Total strip width = 15ft (7.5 bins, rounded to 8 bins)
5. Each bin represents a 2ft × 2ft area
6. Volume calculation sums ALL bins in the strip, not just the center line
